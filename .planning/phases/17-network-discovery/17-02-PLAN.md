    Create `internal/native/t1018_discovery.go` with `//go:build windows` containing:

    1. **ICMP ping sweep — `icmpPingSweep(hosts []string) []string`**:
       - Try `net.ListenPacket("ip4:icmp", "0.0.0.0")` — if error (non-admin per D-09), call `tcpAliveCheck(hosts)` instead
       - When admin: for each host, send hand-crafted ICMP echo request (type=8, code=0):
         ```go
         msg := []byte{8, 0, 0, 0, 0, 0, 0, 1} // type, code, checksum(placeholder), id, seq
         cs := icmpChecksum(msg)
         msg[2], msg[3] = byte(cs>>8), byte(cs)
         ```
       - `conn.SetDeadline(time.Now().Add(300*time.Millisecond))` before each read
       - Collect IPs that respond with ICMP echo reply (type=0)
       - Return slice of alive host IPs

    2. **`icmpChecksum(data []byte) uint16`** — standard RFC 1071 Internet checksum:
       - Sum all 16-bit words, fold carry, complement

    3. **TCP alive check fallback — `tcpAliveCheck(hosts []string) []string`**:
       - For each host, try `net.DialTimeout("tcp4", host+":445", 300ms)` then `:135`
       - If either connects, host is alive — add to result, close conn
       - Use goroutine pool with semaphore (reuse `scanWorkers` concept, 15 workers) for speed

    4. **ARP table dump — `arpTableDump() (string, []arpEntry)`**:
       - `exec.Command("arp", "-a").Output()`
       - Parse output with `parseARPTable(output string) []arpEntry`
       - `type arpEntry struct { IP, MAC string }`
       - Use `strings.Fields(line)` — IP is field[0], MAC is field[1]
       - Skip lines where field[0] doesn't look like an IP (`net.ParseIP(field[0]) == nil`)
       - Return raw output string (for NativeResult.Output) and parsed entries

    5. **nltest DC discovery — `nltestDCDiscovery() string`**:
       - `domain := os.Getenv("USERDNSDOMAIN")` — if empty, use `"."`
       - Try `exec.Command("nltest.exe", "/dsgetdc:"+domain).CombinedOutput()`
       - If exec error (not found): try full path `C:\Windows\System32\nltest.exe`
       - If both fail: return `"nltest: not available (non-domain environment or tool not found)\n"`
       - If command returns non-zero exit (not domain-joined): return `"nltest: DC not found (non-domain environment)\n"`
       - On success: return formatted output including DC name

    6. **DNS reverse lookup — `dnsReverseLookup(hosts []string) string`**:
       - For each host IP: `net.LookupAddr(ip)`
       - If names returned: `fmt.Sprintf("  PTR: %s -> %s\n", ip, strings.Join(names, ", "))`
       - If error: skip silently (no PTR record is normal)
       - Return concatenated results

    7. **`func runT1018() (NativeResult, error)`** — orchestrates all four methods sequentially per D-06:
       - Call `localSubnet()` — if "unknown", return failure result
       - Call `subnetHosts(subnet, localIPAddrs()...)` — exclude own IPs per D-02
       - **Step 1:** ICMP ping sweep (or TCP fallback) — emit header `"=== ICMP Ping Sweep ===\n"`
       - **Step 2:** ARP table dump — emit header `"=== ARP Table ===\n"`
       - **Step 3:** nltest DC discovery — emit header `"=== Domain Controller Discovery ===\n"`
       - **Step 4:** DNS reverse lookup on alive hosts (union of ICMP + ARP IPs) — emit header `"=== DNS Reverse Lookup ===\n"`
       - Concatenate all outputs into `NativeResult{Output: combined, Success: true}`

    8. **`func init() { Register("T1018", runT1018, nil) }`**
