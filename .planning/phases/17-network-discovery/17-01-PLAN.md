    Create `internal/native/t1046_scan.go` with `//go:build windows` containing:

    1. **Package-level constants/vars:**
       - `tcpPorts = []int{21,22,23,25,53,80,135,139,389,443,445,1433,3306,3389,5985,8080,8443}` (per D-04)
       - `udpPorts = []int{53,161,123}` (per D-05)
       - `scanWorkers = 15` (midpoint of D-08 range 10-20)
       - `dialTimeout = 300 * time.Millisecond` (per research recommendation)

    2. **`localSubnet() string`** — duplicate of `detectLocalSubnet()` from engine.go:
       - Iterate `net.Interfaces()`, skip loopback/down
       - Find first IPv4 address, return `fmt.Sprintf("%d.%d.%d.0/24", ip4[0], ip4[1], ip4[2])`
       - Return `"unknown"` on error
       - This avoids import cycle between `internal/engine` and `internal/native`

    3. **`localIPAddrs() []string`** — collect all local IPv4 addresses:
       - Iterate `net.Interfaces()` → `iface.Addrs()` → collect IPv4 strings
       - Used to exclude own IPs from scan results (D-02)

    4. **`subnetHosts(cidr string, excludeIPs ...string) ([]string, error)`**:
       - `net.ParseCIDR(cidr)` — return error if invalid
       - Generate .1 through .254 from the network base
       - Exclude network address (.0), broadcast (.255), and any IP in excludeIPs
       - Return slice of host IP strings

    5. **`type scanResult struct { Host string; Port int; Open bool; Protocol string }`**

    6. **`scanTCP(hosts []string, ports []int) []scanResult`**:
       - Channel semaphore `sem := make(chan struct{}, scanWorkers)` — limits to 15 concurrent goroutines
       - `sync.WaitGroup` for completion tracking
       - Each goroutine: `sem <- struct{}{}` (acquire), defer release, `net.DialTimeout("tcp4", addr, dialTimeout)`
       - If connection succeeds: `conn.Close()`, send `scanResult{Host, Port, true, "tcp"}` to results channel
       - After `wg.Wait()`, close results channel, collect into slice

    7. **`scanUDP(hosts []string, ports []int) []scanResult`**:
       - For each host/port: `net.Dial("udp4", addr)` — always succeeds (connectionless)
       - Send a single null byte, `conn.SetReadDeadline(time.Now().Add(dialTimeout))`
       - Read response — if any data comes back, mark as open
       - UDP results are best-effort; include note in output about Windows Firewall suppression

    8. **`func runT1046() (NativeResult, error)`**:
       - Call `localSubnet()` — if returns "unknown", return `NativeResult{Success: false, ErrorOutput: "cannot detect local subnet"}`
       - Call `localIPAddrs()` to get own IPs
       - Call `subnetHosts(subnet, ownIPs...)` — handle error
       - Emit progress line: `fmt.Sprintf("Scanning %s (%d hosts, %d TCP ports, %d UDP ports)...\n", subnet, len(hosts), len(tcpPorts), len(udpPorts))`
       - Call `scanTCP(hosts, tcpPorts)` — collect open ports
       - Call `scanUDP(hosts, udpPorts)` — collect results, add best-effort note
       - Format results: group by host, list open ports per host
       - Return `NativeResult{Output: output, Success: true}`

    9. **`func init() { Register("T1046", runT1046, nil) }`**
