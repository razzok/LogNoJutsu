🛡️ System Preparation

Configure the system to generate comprehensive Windows logs before simulation. Run all steps once after initial setup.

PowerShell ScriptBlock Logging

Enables Event 4104 (ScriptBlock), 4103 (Module), and Transcription logging.

HKLM\SOFTWARE\Policies\Microsoft\Windows\PowerShell

—

Windows Audit Policy (Best Practice)

Enables 12 audit subcategories: Logon (4624/4625), Process Creation (4688 + cmdline), Account Management (4720/4728), Privilege Use (4672), Scheduled Tasks (4698), Audit Policy Changes (4719).

—

Sysmon Installation

Downloads and installs Sysmon with LogNoJutsu recommended config. Generates: Process (1), Network (3), Registry (12/13), LSASS Access (10), DNS (22).

Requires internet access for automatic download.
